# Настольная книга LLM-инженера: от задачи к рабочему результату

Публичная Markdown-версия русскоязычной настольной книги по LLM-инженерии: от постановки задачи и критериев качества до контекстной инженерии, RAG, инструментов, рабочих процессов, агентов, памяти, проверки качества и доведения LLM-системы до рабочего результата.

## Что это

Это практическое инженерное руководство для работы с LLM-системами. Книга рассматривает модель не как самостоятельный продукт, а как компонент системы, где качество результата зависит от задачи, контекста, источников, инструментов, архитектурных ограничений и проверок.

Основная рамка книги:

```text
задача → критерии качества → контекст → модель → инструменты → проверка → рабочий результат
```

## Для кого

Материал рассчитан на русскоязычных читателей, которым нужно проектировать, оценивать или внедрять прикладные LLM-системы:

- начинающих LLM/AI-инженеров;
- prompt engineers и context engineers;
- разработчиков, которые переходят к AI-инженерии;
- самоучек, собирающих прикладные ИИ-решения через практику;
- предпринимателей и представителей малого и среднего бизнеса, которым нужно понимать применимость, риски и границы LLM-решений.

## Что внутри

Книга состоит из 6 частей, 19 глав и 6 приложений.

Основные темы:

- рабочая рамка LLM-инженера;
- устройство LLM на уровне, достаточном инженеру;
- промпт, контекст, RAG, GraphRAG, инструменты и память;
- постановка задачи и критерии качества;
- рабочие процессы и агенты;
- архитектурный выбор для прикладных ИИ-систем;
- проверка качества, надёжность, безопасность и управление риском;
- переход от демонстрации к рабочему результату;
- внедрение ИИ в бизнес-процесс;
- рабочая среда LLM-инженера.

## Как читать

- Для постановки задачи и выбора подхода: главы 1–4 и 11–12.
- Для проектирования промпта, контекста, RAG или GraphRAG: главы 5–8.
- Для инструментов, рабочих процессов, агентов и памяти: главы 9–10.
- Для качества, надёжности, безопасности и передачи результата: главы 13–16.
- Для внедрения, минимального рабочего контура и личной практики роста: главы 17–19.
